It stores Elements of Same Data Type

Array is used to store the number of elements belonging to same data type. i.e. int a[30]; can store 30 integer variable.

Array Used for Maintaining multiple variable names using single name.

Suppose we need to store 5 roll numbers of students then without declaration of array we need to declare following:
int roll0,roll1,roll2,roll3,roll4;
But it makes program lengthy and complicated like guess if we need to store roll numbers of 100 students then what will be the procedure.so we can use an array named roll i.e.
int roll[5];

Array Can be Used for Sorting Elements

We can store elements to be sorted in an array and then we can sort them by using different sorting technique like insertion sort or bubble sort or any ohter sorting algorithm.

Array Can Perform Matrix Operations

Matrix operations can be performed using the array.We can use 2-D array to store the matrix.

Array Can be Used in Recursive Function

When the function calls another function or the same function again then the current values are stores onto the stack and those values will be retrieve when control comes back. This is similar operation like stack.
