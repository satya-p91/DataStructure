Data structure is a method of organizing large amount of data more efficiently so that any operation on that data becomes easy. More precisely, a data structure is a collection of data values, the relationships among them, and the functions or operations that can be applied to the data. Based on organizing mathod , data structures are divided into two types.

1.  Linear Data Structures

Data structure that organize the data in sequential order, then that data structure is called as Linear Data Structure

  • Arrays
  • Linked list
  • Stack
  • Queue, etc.

2.  Non-Linear Data Structures

Data structure that organize the data in random order, then that data structure is called as Non-Linear Data Structure.

  • Tree
  • Graph
  • heaps , etc.

Basic Terminology

  • Data − Data are values or set of values.
  • Data Item − Data item refers to single unit of values.
  • Attribute and Entity − An entity is that which contains certain attributes or properties, which may be assigned values.
